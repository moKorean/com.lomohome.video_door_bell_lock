이 앱은 비디오 초인종을 포함한 RTSP IP 카메라를 스마트 도어락, 모션·초인종 센서와 함께 하나의 도어 화면으로 묶어줍니다. 카메라를 하나 또는 여러 대 추가하면 클라우드 없이, 화질 변환 없이 라이브 영상을 한눈에 볼 수 있습니다. 로컬 네트워크에서 카메라를 자동으로 검색해 주므로, 대개 스트림 주소만 한 번 채워 넣으면 됩니다. RTSP를 지원하는 비디오 도어벨·카메라와 도어락을 함께 사용할 수 있습니다.

도어락과 초인종 센서, 모션 센서는 모두 선택 사항이라, 카메라만 추가한 기기는 영상만 보여주고, 모두 연동한 기기는 센서 상태와 잠금·해제 버튼까지 같은 화면에서 함께 보여줍니다. 이를 바탕으로 자동화를 만들 수 있습니다. 누군가 초인종을 누르면 카메라를 확인하고 문을 열거나, 움직임이 감지되면 현관 등을 켜고 녹화를 시작하거나, 밤에는 문을 자동으로 잠그는 식입니다. 연동한 기기는 언제든지 나중에 바꿀 수 있고, 화면은 사용하는 언어를 따라 표시됩니다. 모든 동작은 로컬에서 처리되어 사용자가 직접 제어합니다.
